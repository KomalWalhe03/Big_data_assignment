# Big Data Assignment - Stock Market Analysis

## Overview
This project analyzes stock market data using Python, Pandas, and Streamlit.  
It includes data cleaning, aggregation, and interactive visualizations.

## Files
- `data_cleaning.py` : Cleans and normalizes raw CSV data
- `streamlit_app.py` : Streamlit app for visualizing aggregates
- `cleaned.parquet` : Cleaned dataset
- `agg1.parquet`, `agg2.parquet`, `agg3.parquet` : Aggregated data
- `requirements.txt` : Python dependencies

## How to Run
1. Clone the repository  
2. Create a virtual environment: `python -m venv venv`  
3. Activate the environment and install dependencies: `pip install -r requirements.txt`  
4. Run Streamlit app: `streamlit run streamlit_app.py`
